Sobre o Evento

O Pylestras é um evento sem fins lucrativo que consiste em um​ dia de palestras de tecnologia no qual palestrante e os convidados e/ou selecionados por votação compartilham sua experiência com estudantes e profissionais da área. Os assuntos das palestras são variados, contudo, orbitam o ecossistema da linguagem de programação​ Python.

Estrategicamente, procuramos manter um slot​ de palestra para uma linguagem de​ programação convidada, que varia entre os eventos, visando integrar o PugCE com outras comunidades de tecnologia do estado. Convidamos vocês para nossa 17ª edição, que ocorrerá em breve!

Comunidade: muito mais do que um grupo de desenvolvedores

Ryllari Santana

As comunidades tecnológicas são uma atmosfera incrível para o desenvolvimento pessoal e profissional. Frequentemente participamos de comunidades com a visão de que estamos apenas "incluídos em um grupo de desenvolvedores".

O objetivo desta palestra é trazer uma visão do que uma comunidade realmente significa, os esforços que ela precisa para sobreviver e os desafios e benefícios que existem em ser um participante ativo, despertando a ideia de que isso é algo feito por todos e para todos. todos.

Se no final da palestra você não se sentir desafiado a fazer mais pelas comunidades a que pertence, minha meta não foi alcançada.

Python e Lisp: como unir dois mundos

Eliseu Lucena Barros

Python é o remanescente das linguagens interpretadas, a linguagem favorita para lidar com Machine Learn e Big Data, porem, a um tempo atras, essa realidade era ocupada por outra figura, Lisp, Lisp foi criado justamente para lidar com Machine Learn nos Laboratórios de pesquisa em Inteligência Artificial do MIT, isso na década de 60, mas como uma linguagem antiga, mais antiga que COBOL e VB pode ter sua relevância hoje em dia?

Bem, seu apurado sistema de macros, alta facilidade de modificação da linguagem, abordagem preferencialmente Funcional e o Orientado a Objetos, Syntax enxuta e simplificada, Tipagem Fraca, Interpretada e a característica de ter uma linha tênue entre o que é dado e o que é código, entre muitos outros, a tornam a melhor ferramenta para resolver uma gama muito grande de problemas

Mas você pode estar se perguntando como Lisp pode se relacionar com Python, um projeto quer unir o melhor dos dois mundos, uma nova linguagem chamada Hylang pretende rodar Lisp encima da VM do Python, e é isso que iremos discutir

Python e Programação Funcional? O que diabos é isso?

Rebeca Maia Pontes

Python é uma linguagem de alto nível muito versátil, permitindo múltiplos paradigmas incluindo o paradigma funcional. Esse paradigma foi criado para facilitar o uso de funções matemáticas na computação e é utilizado em situações onde é preciso executar muitas operações no mesmo conjunto de dados. É usado em Machine Learning, Big Data, dentre outras aplicações. Embora Python não seja puramente funcional, ele é muito utilizado pela sua simplicidade e facilidade de uso, se comporada a outras linguagens funcionais como Haskell. Nessa palestra será mostrado alguns exemplos de como Python pode ser aplicado em programação funcional e seus benefícios.

Python, Machine Learning e Pokemons. Por que não ?!

Alexssandro Oliveira

Que tal aprender sobre Machine Learning com Python colocando Pokemons pra batalhar? O objetivo dessa palestra é de maneira simples conhecer o ecossistema que o Python proporciona para ciencia de dados , entender os desafios que isso acompanha e quem sabe ter insights de onde aplicar.

Python - Principais recursos e áreas de aplicação

Weberty Silveira

É notável o crescimento da linguagem python nos últimos anos, mas essa velocidade com que vem ganhando espaço e se tornando rapidamente uma das linguagens preferidas do mercado, não é por acaso.

O Python apresenta diversas bibliotecas de desenvolvimento, que permitem aumentar a produtividade dos mais experientes. Além disso, a sua sintaxe simples oferece uma aprendizagem divertida e rápida para novos programadores.

Nessa palestra poderão conferir as facilidades que a linguagem oferece, além de conhecermos um pouco do que ela é capaz de realizar nas mais diversas áreas de desenvolvimento.
